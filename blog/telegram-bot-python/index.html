Telegram Bot for Python

Redirecting my post in LinkedIn (Click Here), I want to retype it to my GitHub blog.

I want to share a free simple script for helping us to announce what is our python program doing in our telegram group chat. It’s only changing ‘YOUR TOKEN’ and ‘YOUR CHAT ID’ in the script. Finally, yup, all including functions can be used by inputting these in our own programs/scripts. These are some functions in this script:

bot_sendText('Your message') #to send a string message
bot_sendPhoto('Your Photo Path') #to send an output figure
bot_sendDocument('Your Document Path') #to send a document (sheet, doc, etc)

Requirement

installing request package

How to do

YOUR TOKEN could be obtained by:

creating your account bot in @BotFather -> send /newbot -> following the next instruction.

YOUR CHAT ID could be obtained by these instructions:

  1. Add the Telegram BOT to your destination group.

  2. Get the list of updates for your BOT (in there, you copy chat id to the script):

api.telegram.org/bot(YourBOTToken)/getUpdates

Happy Coding! If you have some ideas and suggestions, please give me some feedback. Thank you!
